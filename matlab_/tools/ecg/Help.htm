ECG Beat Fitter Help

1- Select the critical ECG points by clicking on the ECG plot.

2- Click Optimize to start the optimization of the parameters.

3- The number of Gaussian kernels used for the optimization and the fitting error is reported in the report panel.

4- Click Export Parameters to export the parameters to the work space. The exported parameters are stored in a vector called OptimumParams or with the name defined by the calling function.

5- You can change the selected critical points by clicking on the Reset button.
